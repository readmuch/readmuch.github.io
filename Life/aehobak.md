# 고소하게 볶은 애호박 나물

2인분 기준 기본 재료와 불 조절 타이밍만 잡아도 무난히 맛이 나오는 레시피예요. 밥상에 한 그릇 올리기 좋은 담백한 반찬!

## 재료
- 애호박 1개(약 200g), 반달썰기
- 양파 1/4개, 채썰기
- 식용유 1큰술
- 다진 마늘 1/2작은술
- 새우젓 1작은술
- 참치액젓 또는 멸치액젓 1작은술
- 참기름 1작은술
- 통깨 1작은술

## 조리 순서
1. 팬을 중불로 달군 뒤 식용유 1큰술과 다진 마늘 1/2작은술을 넣어 향이 날 때까지 살짝 볶는다.
2. 애호박과 양파를 넣고 중불에서 1~2분 볶아 숨을 죽인다.
3. 새우젓 1작은술과 액젓 1작은술을 넣고 애호박이 반투명해질 때까지 3~5분 더 볶는다.
4. 불을 끄고 참기름 1작은술과 통깨 1작은술을 넣어 고루 섞은 뒤 그릇에 담는다.

## 맛있게 만드는 팁
- **간 맞추기**: 새우젓의 짠맛이 다르니 중간에 한 조각 맛을 보고 필요하면 물 약간을 더하거나 액젓을 조금씩 추가한다.
- **식감 살리기**: 애호박을 너무 오래 볶으면 물러지므로 반투명해지는 순간 불을 끄고 여열로 마무리한다.
- **대체 재료**: 액젓 대신 소금 1/3작은술을 먼저 넣고 모자라면 더한다. 매콤하게는 청양고추 1개를 함께 볶아도 좋다.

## 간단 요약
- 비율: 애호박1 / 양파¼ / 새우젓1t / 액젓1t / 참기름1t / 통깨1t / 마늘½t / 기름1T
- 순서: 마늘향 내기 → 애호박·양파 1~2분 → 새우젓·액젓 후 3~5분 → 불 끄고 참기름·통깨
