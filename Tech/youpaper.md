# 유페이퍼 전자책 셀프 출판 

## 1. 사전 준비: 도구와 폰트 세팅

가장 먼저 '재료'를 준비해야 합니다. 특히 저작권 문제가 없는 폰트 사용은 필수입니다.

### **① 상업용 무료 폰트 (KoPub 바탕체) 설치**

전자책은 '상업적 판매'가 목적이므로 시스템 기본 폰트(굴림, 돋움 등)보다는 전자출판 전용 무료 서체를 써야 안전합니다.

* **다운로드:** [한국출판인회의 서체 페이지](https://www.google.com/search?q=http://www.kopus.org/Biz/electronic/Font.aspx)에 접속하여 **KoPub World 서체**를 다운로드합니다.
* **설치 방법:** 압축을 푼 뒤, 윈도우 사용자라면 **TTF** 파일을 마우스 우클릭하여 '모든 사용자용으로 설치'를 클릭합니다.
* **활용법:** 원고 작성(워드, 한글 등) 시 본문 전체를 **'KoPub World 바탕체 Light 또는 Medium'**으로 설정하세요. 눈의 피로도가 훨씬 줄어듭니다.

### **② 표지 이미지 제작**

* **규격:** 가로 600px × 세로 900px 이상 (비율 1:1.5)
* **방법:** '캔바(Canva)'나 '미리캔버스'에서 **전자책 표지** 템플릿을 선택해 제목과 저자명만 수정하면 훌륭한 표지가 완성됩니다.

## 2. 전자책 검수 기준 (반려 방지 필수 지식)

유페이퍼 운영진이 책을 승인할 때 확인하는 기준입니다. 이 내용을 어기면 승인이 거절될 수 있습니다.

### **[공통 기준] 모든 전자책에 반드시 들어가야 할 내용**

* **표지:** 책의 얼굴입니다. 제목과 저자명이 명확해야 합니다.
* **판권 정보(Copyright):** 책의 맨 뒤(혹은 맨 앞)에 반드시 포함되어야 합니다. (아래 항목 포함)
* 도서명 / 저자명 / 발행일 / 발행처(유페이퍼 혹은 개인)
* 무단 전재 및 복제를 금한다는 저작권 문구
* ISBN 번호가 들어갈 자리 (발급 전이라면 'ISBN : 예정'으로 기입)


* **콘텐츠 품질:** 오타가 너무 많거나, 내용이 지나치게 부실한 경우, 혹은 타인의 글을 무단 복제한 경우 반려됩니다.

### **[PDF] 검수 포인트**

* **폰트 임베딩(Embedding):** 가장 중요한 부분입니다. PDF 파일 안에 사용한 폰트가 정보로서 박혀 있어야 합니다. (3번 항목에서 해결법 설명)
* **여백 설정:** 상하좌우 여백이 너무 적으면 기기에서 읽을 때 글자가 잘려 보일 수 있습니다. (사방 15~20mm 권장)
* **가독성:** 글자 크기가 너무 작거나 배경색과 글자색의 대비가 낮아 읽기 힘들면 안 됩니다.

### **[EPUB] 검수 포인트**

* **논리적 목차(TOC):** EPUB은 기기에서 목차 버튼을 눌렀을 때 해당 페이지로 바로 이동해야 합니다. 뷰어 자체 목차가 구성되어야 승인됩니다.
* **이미지 최적화:** 삽입된 이미지 한 장의 용량이 너무 크면(예: 5MB 이상) 책이 넘어갈 때 끊김이 발생해 반려될 수 있습니다.
* **문단 정렬:** 글자 크기를 키워도 문단이 깨지지 않도록 '들여쓰기'나 '줄 간격' 설정을 깔끔하게 유지해야 합니다.

## 3. PDF-XChange Editor로 PDF 편집 및 폰트 박기

원고를 PDF로 만든 후, 유페이퍼에 올리기 직전 '최종 점검' 단계입니다.

1. **오타 수정하기:** `편집(Edit)` 메뉴 -> `콘텐츠 편집(Edit Content)`을 누르면 이미 만들어진 PDF 내의 글자를 직접 고칠 수 있습니다.
2. **페이지 관리:** `구성(Organize)` 메뉴에서 표지를 맨 앞에 끼워 넣거나, 잘못 들어간 빈 페이지를 삭제할 수 있습니다.
3. **폰트가 제대로 포함됐는지 확인법:**
* `파일(File)` -> `문서 속성(Document Properties)` -> `글꼴(Fonts)` 탭을 클릭합니다.
* 리스트에 있는 모든 글꼴 옆에 **(Embedded Subset)** 문구가 있는지 확인하세요.


4. **폰트 강제 포함시키기 (비법):**
* 만약 글꼴이 포함되지 않았다면, 당황하지 마세요. `인쇄(Ctrl+P)`를 누릅니다.
* 프린터 대상을 **'PDF-XChange Printer'**로 선택하고 다시 저장하세요. 이 과정을 거치면 KoPub 바탕체 등 사용된 폰트가 자동으로 PDF 파일 안에 박히게 됩니다.

## 4. 유페이퍼 출판 순서 (실전 단계)

### **1단계: 판매자 등록**

* 유페이퍼 로그인 후 **[마이페이퍼] > [판매자 정보 설정]**에서 정산받을 계좌 번호를 등록하고 본인 인증을 마칩니다.

### **2단계: 나의 페이퍼 생성**

* 유페이퍼 내에 나만의 작은 서점인 '페이퍼'를 만듭니다. 서점 이름과 소개글을 작성하세요.

### **3단계: 전자책 등록**

* **정보 입력:** 도서명, 카테고리, 저자명, 판매 가격을 입력합니다.
* **ISBN 신청:** 유페이퍼에서 제공하는 'ISBN 발급 대행' 서비스를 체크하세요. (약 1,000~2,000원 비용 발생)
* **파일 업로드:** 준비한 표지(JPG)와 원고(PDF 또는 EPUB)를 업로드합니다.

### **4단계: 판매 승인 요청**

* [등록] 버튼을 누르면 유페이퍼 운영진이 검수를 시작합니다.
* **영업일 기준 1~3일** 정도 소요되며, 검수가 완료되면 카카오톡이나 메일로 승인 알림이 옵니다. 이때부터 교보문고, 예스24 등 제휴 서점으로 유통이 시작됩니다.
